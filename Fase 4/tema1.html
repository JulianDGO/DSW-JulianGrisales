    • ¿QUE ES CSS?

    La hoja de estilo en cascada (Css) es un lenguaje de diseño el cual actualmente esta en su version de CSS3 la cual tiene mejoras en la estruturación de codigo,se centra principalmente en la apariencia de cualquier documento que este unido con un lenguaje de hipertexto. La apariencia tiene que ver con el tipo de letra, espacios, cajas,etiquetas entro otros, las cuales se pueden crear por separado.
    CSS fue creado para ayudar al HTML ya que este no fue diseñado para la creación de etiquetas, y las que tenia daban mas problemas que beneficio.

    Ventajas de Css

    - Las diferentes hojas de estilo creado se pueden utilizar en diferentes dispositivos o creación de otras páginas web.
    - Se pueden almacenar diferentes estilos por separado de una página web para una especificación de diseño diferentes lo cual, permite tanto compartir o reutilizar tus diseños en otros proyectos.
    - Durante la carga de la página web es mucho menor, ya que al obtener formatos de CSS todo almacenamiento de este queda como cache y por ende la banda ancha al ejecutar la página es mucho más rápida.

    Desventaja de CSS

    - Las modificaciones por medio de hojas de estilo de reset agrega peso a la página aumentando la banda ancha de la página web y por ende en tiempo de carga aumenta.
    - Cada reset tiene sus características las cuales modifica la página web que utilicemos, pero al colocarlo en otro código puede chocar con otros reset que ya estén colocados, por ende, produces errores al cargar las páginas web
