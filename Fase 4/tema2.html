    ¿QUE ES FRAMEWORK CSS?

    Es una Plantilla la cual facilita la labor de programación ofreciendo características y funciones mediante el mejoramiento del proceso de programación reduciendo errores y consiguiendo una página muy buena

    VENTAJAS DE FRAMEWORK

    1. Agiliza el proceso de Desarrollo Un Framework Css nos facilita desarrollar proyectos de manera rápida mediante un código Base el cual ya tiene definida dimensiones y el diseño de cómo va a quedar facilitando nuestro trabajo.
    2. Facilidad Grupal El Framework css al tener un código base nos facilita la creación de aplicaciones grupales ya que todos tienen el mismo código, lo cual al trabajar de tal manera el aplicativo van a lucir igual en todos los integrantes, lo que es realmente importante es la personalización del contenido.

    DESVENTAJAS DE FRAMEWORK

    1. Falta de personalidad. Al tener un código base las modificaciones visuales son siempre las misma lo cual para proyectos personalizados no se pueden utilizar, ya que tiene una serie de medidas, selectores y etiquetas las cuales no podrán ser modificadas.
    2. Pérdida de tiempo Al utilizar un Frameworks ya creado tiene que saber como manejarlo para sacarle la ventaja al máximo, por ende si ya sabe programar puede utilizar el tiempo que se demorara en aprender y crear su propio código para el proyecto que vas a hacer.
