    Hojas de Estilo en Cascada CSS

    Css o mejor llamado "Hojas de estilo en cascada" es el lenguaje el cual maneja todo el diseño de las paginas web conjunta con el leguanje de html, el estilo en cascada es un proceso el cual lleva un proceso descendiente lo que quiere decir que va de arriba hacia abajo, durante la digitación toca tener en cuenta la Especificidad de los componentes que implementaremos en el codigo de CSS ya que determinara la importacia de cual se activará primero antes que otro siempre teniendo en cuenta el orden es cascada. Este es el orden de la estructurado de Especificidad dentro de Css:

    1. !Important
    2. Atributo Style
    3. El Id
    4. Clases, pseudoclase o atributos
    5. Elementos o Pseudoelementos
    Durango, A. (2015). Diseño Web con CSS: 2ª Edición. IT Campus Academy.
    https://revistas.udistrital.edu.co/index.php/reving/article/view/2311

    Framework CSS

    Un Framework de Css son bibliotecas web que tiene diferentes estilos ya diseñados los cuales pueden ser implementados en los diferentes sitios web que vayamos a desarrollar, la funcionalidad de cada uno es muy amplia pero tambien quiere decir que es muy utilizada por ende las paginas web con la misma librerias se pareceran, la falta de aprovechamiento completamento de la libreria seleccionada hace que nuestra pagina web quede muy pesada lo cual conlleva a posibles demoras al abrir el archivo. Algunos Framework en CSS son:

    1. 960 Grid System
    2. Blueprint Css
    3. Bulma
    4. Bootstrap
    Aponte, Á. M. V. (2014). Guía comparativa de Frameworks para los lenguajes HTML 5, CSS y JavaScript para el desarrollo de aplicaciones Web (Doctoral dissertation, Universidad Tecnológica de Pereira.
