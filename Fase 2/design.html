    Comprender las diferentes estrategias de apendizaje y la importacia de la utilzación del orden en Cascada al utilizar diferentes estilos en CSS y como las diferentes bibliotecas de Framewords Css hace que ese trabajo sea mucho mas rapido a la hora de programar.

    Orden en Cascada

    El orden en Cascada es un proceso el cual lleva un ordenamientos desde arriba hacia abajo en la linea de codigo, es muy importante en la implementación de estilos Css ya que el orden determina como va a funcionar las diferentes ordenes presentada por el programador, pero tambien que tener en cuenta la Especificidad de los diferentes componentes ya que estos hace que se conosca cual es la importancia a la hora programar para asi saber cual funcionara y cual no. Algunos ejemplos de como estra estructurado la Especificidad dentro del orden en cascada es el siguiente:

    1. Atributo Style
    2. El Id
    3. Clases, pseudoclase o atributos
    4. Elementos o Pseudoelementos
    Durango, A. (2015). Diseño Web con CSS: 2ª Edición. IT Campus Academy.
    https://revistas.udistrital.edu.co/index.php/reving/article/view/2311

    Framework CSS

    Un Framework de Css es una biblioteca de diferentes estilos ya diseñados los cuales pueden ser implementados en los diferentes sitios web que estemos haciendo, es facil agregarlos y la funcionalidad de cada uno es muy amplia pero eso quiere decir que puede verse como cualquier pagina que tambien tenga implementado el mismo frameword, al no aprovechar completamente la libreria seleccionada hace que nuestra pagina web quede muy pesada lo cual colleva a posibles demorar al abrir el archivo. Algunos de los diferentes Framework en CSS son:

    1. 960 Grid System
    2. Blueprint Css
    3. Bulma
    4. Bootstrap
    Aponte, Á. M. V. (2014). Guía comparativa de Frameworks para los lenguajes HTML 5, CSS y JavaScript para el desarrollo de aplicaciones Web (Doctoral dissertation, Universidad Tecnológica de Pereira.
